# TomiFit

Sitio web estático para Tomás Agustín Miguel – Asesoría Élite.

## Estructura

- `index.html` – Web principal.
- `style.css` – Estilos.
- `assets/logototo-.png` – Subí tu logo en esta carpeta tras subir el repo.

## Publicación en Vercel

1. Subí este proyecto a un repo “TomiFit” en GitHub.
2. Creá una cuenta en [Vercel](https://vercel.com/), conectá tu GitHub e importá el repo.
3. Esperá la build automática y ¡tu web estará publicada gratis!
4. (No olvides subir tu logo a la carpeta correcta.)
